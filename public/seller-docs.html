üõçÔ∏è Mahsulot Boshqaruvi

To'liq mahsulot boshqaruvi tizimi

GET Mahsulotlar Ro'yxati
/api/seller/products?page=1&limit=10&search=shirt&category=clothing&status=active&sortBy=newest&language=uz

Parametrlar:

  • page - Sahifa raqami (default: 1)
  • limit - Sahifadagi elementlar soni (default: 10)
  • search - Qidiruv so'zi
  • category - Kategoriya nomi
  • status - active, inactive, all
  • sortBy - newest, oldest, name_asc, name_desc, price_asc, price_desc
  • language - Til kodi (uz, ru, en)
GET Bitta Mahsulot
/api/seller/products/:id?language=uz

üì¶ Buyurtma Boshqaruvi

Buyurtmalarni kuzatish va holat yangilash

GET Buyurtmalar Ro'yxati
/api/seller/orders?page=1&limit=10&status=pending&search=customer&sortBy=newest

Parametrlar:

  • status - all, pending, processing, shipped, delivered, cancelled
  • search - Mijoz nomi yoki telefon raqami
  • startDate - Boshlanish sanasi (YYYY-MM-DD)
  • endDate - Tugash sanasi (YYYY-MM-DD)
  • sortBy - newest, oldest, amount_high, amount_low
GET Buyurtma Statistikalari
/api/seller/orders/statistics
GET Bitta Buyurtma
/api/seller/orders/:id
PATCH Buyurtma Holati Yangilash
/api/seller/orders/:orderId/items/:itemId/status

So'rov tanasi:

{ "status": "shipped", "trackingNumber": "TRK123456789", "notes": "Shipped via DHL" }

Mavjud statuslar:

  • processing - Ishlanmoqda
  • shipped - Yuborildi
  • delivered - Yetkazildi
  • cancelled - Bekor qilindi

// Seller dashboard ma'lumotlarini olish const getSellerDashboard = async () => { try { const response = await fetch('/api/seller/dashboard', { method: 'GET', headers: { 'Authorization': `Bearer ${sellerToken}`, 'Content-Type': 'application/json' } }); const data = await response.json(); console.log('Dashboard data:', data); } catch (error) { console.error('Error:', error); } }; // Mahsulot yaratish (multi-language) const createProduct = async (productData) => { const formData = new FormData(); // Mahsulot ma'lumotlarini qo'shish formData.append('name', productData.name); formData.append('description', productData.description); formData.append('categoryId', productData.categoryId); formData.append('variants', JSON.stringify(productData.variants)); // Tarjimalarni qo'shish formData.append('translations', JSON.stringify(productData.translations)); // Rasmlarni qo'shish productData.images.forEach(image => { formData.append('images', image); }); try { const response = await fetch('/api/seller/products', { method: 'POST', headers: { 'Authorization': `Bearer ${sellerToken}` }, body: formData }); const result = await response.json(); console.log('Product created:', result); } catch (error) { console.error('Error:', error); } }; // Mahsulot tarjimasini yangilash const updateProductTranslations = async (productId, translations) => { try { const response = await fetch(`/api/seller/products/${productId}/translations`, { method: 'PUT', headers: { 'Authorization': `Bearer ${sellerToken}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ translations }) }); const result = await response.json(); console.log('Translations updated:', result); } catch (error) { console.error('Error:', error); } }; // Foydalanuvchi til va valyuta sozlamalarini yangilash const updateLanguageCurrency = async (language, currency) => { try { const response = await fetch('/api/users/language-currency', { method: 'PUT', headers: { 'Authorization': `Bearer ${sellerToken}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ preferredLanguage: language, preferredCurrency: currency }) }); const result = await response.json(); console.log('Language/Currency updated:', result); } catch (error) { console.error('Error:', error); } }; // Variant yaratish const createVariant = async (variantData) => { try { const response = await fetch('/api/variants', { method: 'POST', headers: { 'Authorization': `Bearer ${sellerToken}`, 'Content-Type': 'application/json' }, body: JSON.stringify(variantData) }); const result = await response.json(); console.log('Variant created:', result); } catch (error) { console.error('Error:', error); } }; // Buyurtma holati yangilash const updateOrderStatus = async (orderId, itemId, status, trackingNumber) => { try { const response = await fetch(`/api/seller/orders/${orderId}/items/${itemId}/status`, { method: 'PATCH', headers: { 'Authorization': `Bearer ${sellerToken}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ status, trackingNumber, notes: 'Status updated by seller' }) }); const result = await response.json(); console.log('Status updated:', result); } catch (error) { console.error('Error:', error); } };
